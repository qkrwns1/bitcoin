# Offline Data Layout

- `raw/candles_1s`: Upbit 1-second candle raw parquet files
- `raw/trades`: Upbit trade tick raw parquet files
- `processed/features`: feature tables for training or research
- `processed/episodes`: RL episode datasets
- `db`: local DuckDB database files
- `models`: trained model artifacts
- `logs`: training and data pipeline logs
